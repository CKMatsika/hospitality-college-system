Student Endpoints

GET /api/student/profile

Get student profile information

Headers: Authorization: Bearer {token}

GET /api/student/grades

Get student grades and academic performance

Headers: Authorization: Bearer {token}

GET /api/student/attendance

Get student attendance records

Headers: Authorization: Bearer {token}

GET /api/student/fees

Get fee information and payment status

Headers: Authorization: Bearer {token}

GET /api/student/schedule

Get class schedule and timetable

Headers: Authorization: Bearer {token}

Quick Start Guide

1. Get Token

Login to get your API token

2. Make Requests

Include token in Authorization header

3. Access Data

Use endpoints to get student data

4. Build App

Create mobile app or integration

Response Codes

200

Success - Data returned

401

Unauthorized - Invalid token

403

Forbidden - No permission

404

Not Found - Resource missing
